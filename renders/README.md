# GModular Render Outputs

This directory contains render verification images generated by the tools in `../tools/`.

## Render Scripts

| Script | Purpose |
|--------|---------|
| `tools/render_verification.py` | Multi-angle verification renders (iso, south, topdown, interior, NW) |
| `tools/render_final.py` | High-quality final renders with Phong lighting and proper textures |
| `tools/render_textured.py` | UV-mapped textured renders using actual KotOR TGA textures |

## Naming Convention

- `final_*.png` / `final_*.ppm` — Final quality renders (v3 pipeline)
- `v2_*.png` / `v2_*.ppm` — Version 2 renders (improved Phong lighting)
- `render_out_*.png` — Output from render_verification.py
- `test_render*.ppm` — Quick test renders

## Usage

```bash
# Generate all verification renders (requires a .mod file):
python tools/render_verification.py path/to/module.mod

# Generate high-quality final renders:
python tools/render_final.py path/to/module.mod

# Generate textured renders with UV mapping:
python tools/render_textured.py path/to/module.mod
```

> **Note**: Render output images (*.png, *.ppm) are excluded from git via .gitignore.
> Only this README and the render scripts are tracked.
